# Obsidian

Obsidian is a local-first markdown knowledge base. It treats any folder of plain
markdown files as a "vault" and renders `[[wikilinks]]` between notes as a navigable
graph. It is the editor the wiki output is designed to be read in.

## Overview

Obsidian stores everything as local files — no proprietary database, no required cloud
sync. Notes are plain markdown, so they remain readable and portable outside the app.

## Key Facts

- A vault is just a directory; opening a folder makes it editable in Obsidian.
- The graph view visualizes notes as nodes and `[[wikilinks]]` as edges.
- Community plugins such as Dataview extend Obsidian with frontmatter-scoped queries.

## Origin

Obsidian is developed by Dynalist Inc. and was first released in 2020. It is free for
personal use.
