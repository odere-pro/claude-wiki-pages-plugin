# Obsidian

## Overview

A local-first markdown editor that treats a folder of `.md` files as a vault.
Used to read and navigate the wiki, including its graph view. ([[Getting Started]])

## Key Facts

- A vault is just a folder on disk. ([[Getting Started]])
- Plugin configuration is documented further in [[Ghost Page]].
