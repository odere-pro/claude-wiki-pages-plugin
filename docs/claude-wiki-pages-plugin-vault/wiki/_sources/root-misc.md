# Root Miscellaneous Files

## Metadata

- **Files**: `raw/repo/root/LICENSE`, `raw/repo/root/NOTICE`, `raw/repo/root/THIRD_PARTY_LICENSES.md`, `raw/repo/root/SUPPORT.md`, `raw/repo/root/CODE_OF_CONDUCT.md`, `raw/repo/root/install-macos.sh`
- **Scope**: Root governance, license, and support files
- **Type**: Community and legal governance

## Summary

Root-level governance files covering the Apache-2.0 license, third-party attribution, community support posture, code of conduct (Contributor Covenant v2.1), and the macOS one-command dependency installer.

## Key Claims

`LICENSE`: Apache-2.0. `NOTICE`: third-party attribution for bundled MIT-licensed kepano/obsidian-skills. `THIRD_PARTY_LICENSES.md`: full license texts. `SUPPORT.md`: best-effort community support, no SLA. Bug reports go to GitHub Issues with schema_version + plugin version + minimal fixture. Severity levels: Critical (hotfix), High (next release), Medium/Low (welcomed PRs). Out-of-scope for support: Obsidian, Claude Code, kepano/obsidian-skills, user-authored vault content. `CODE_OF_CONDUCT.md`: Contributor Covenant v2.1, pledges harassment-free environment, enforcement via maintainer email. `install-macos.sh`: idempotent one-liner that installs Homebrew, git, jq, and Bun and patches PATH.
Covers: License, Support, Code of Conduct, macOS Install, Third-Party Attribution
