# User Guide 01: Getting Started

## Summary

Prerequisites, scaffold vault, doctor command, Obsidian setup. The day-1 path from install to first cited answer.

## Key Claims

- Prerequisites: Claude Code ≥ 2.0, bash + git + jq on PATH, Bun ≥ 1.2 (recommended).
- Install: `/plugin marketplace add odere-pro/claude-wiki-pages-plugin` → `/plugin install claude-wiki-pages` → `/claude-wiki-pages:init`.
- The onboarding wizard scaffolds `vault/` from `docs/vault-example/`.
- `/claude-wiki-pages:doctor` is the health check after install; exit 0 = OK.
- Obsidian: open the vault folder, install Dataview community plugin for the dashboard.
