# Getting Started (CLI Quickstart)

## Summary

CLI quickstart: install, init, import files, wiki command, status, query, markdown export. A condensed path from install to first cited answer.

## Key Claims

- Five steps: install plugin → init vault → drop sources into `raw/` → `/claude-wiki-pages:wiki` → `/claude-wiki-pages:query "your question"`.
- Status check: `/claude-wiki-pages:status` — green = clean ingest.
- Export: `/claude-wiki-pages:markdown` — renders a query answer as portable markdown in `vault/output/`.
- The vault is a directory the user controls; the plugin is context loaded by Claude Code.
