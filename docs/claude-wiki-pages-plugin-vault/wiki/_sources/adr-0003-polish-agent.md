# ADR-0003: Polish Agent and Obsidian-Side Experience

## Summary

Establishes the polish agent (`claude-wiki-pages-polish-agent`) as the tail-of-write step that keeps the Obsidian-side experience in sync after every ingest or curator pass. The polish agent owns graph colors, vault MOC regeneration, and per-folder folder-note consistency.

## Key Claims

- The polish agent runs automatically after every ingest or curator pass.
- Owns three idempotent steps: (1) apply graph colors for new top-level topic folders; (2) regenerate `wiki/index.md` from per-folder folder notes; (3) reconcile every folder note's children/child_indexes against filesystem siblings.
- Amended by ADR-0022 for headless fallback when Obsidian CLI is unavailable.
- Amended by ADR-0023: the layer pass is dropped; canonical group order is now topics → specials.
