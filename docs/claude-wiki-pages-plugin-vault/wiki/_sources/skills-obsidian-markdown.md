# Obsidian Markdown Skill — SKILL.md

## Metadata

- Source: `raw/repo/skills/obsidian-markdown/SKILL.md`
- Type: MIT-licensed Obsidian reference skill (kepano/obsidian-skills)

## Summary

The `obsidian-markdown` skill documents Obsidian Flavored Markdown syntax extensions — wikilinks, embeds, callouts, properties, comments — beyond standard CommonMark and GitHub-Flavored Markdown. A general-purpose reference with no plugin-specific conflicts.

## Key Claims

Covers: Obsidian Markdown Skill, Wikilink Syntax, Embed Syntax, Callout Syntax, Properties Frontmatter, Tag Syntax.

Wikilinks: `[[Note Name]]`, `[[Note Name|Display Text]]`, `[[Note Name#Heading]]`, `[[Note Name#^block-id]]`. Block ID: append `^my-block-id` to any paragraph; for lists/quotes, place the block ID on a separate line after the block. Embeds: prefix any wikilink with `!` to embed inline. Callout types include `note`, `tip`, `warning`, `info`, `example`, `quote`, `bug`, `danger`, `success`, `failure`. Frontmatter default properties: `tags` (searchable labels), `aliases` (alternative note names for link suggestions), `cssclasses`.
