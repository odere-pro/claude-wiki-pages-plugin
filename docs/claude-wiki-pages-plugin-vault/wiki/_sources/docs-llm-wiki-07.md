# LLM Wiki Guide 07 — Query the Wiki

## Summary

Guide 07 explains how to query the wiki: using `/claude-wiki-pages:wiki` with a question, the analyst agent's five operating modes (Query, Dashboard, Document Compile, Extract, Challenge), the challenge framing for adversarial self-questioning, and the six-step query protocol (read index → find pages → follow links → synthesize → cite → offer synthesis note).

## Key Claims

Query entry: `/claude-wiki-pages:wiki <question>`. Analyst agent modes: Query (answer with citations), Dashboard (compile metrics), Document Compile (reconstruct a full doc from pages), Extract (pull structured data), Challenge (adversarial — find contradictions, gaps, past decisions arguing against the current plan). Challenge framing template: "I'm about to decide X. Search for past decisions, contradictions, evidence gaps, arguments against." Every answer ends with a Sources section (research-paper style, numbered, one entry per consulted page). Novel answers may be filed as synthesis notes.

Covers: Wiki Query, Analyst Agent Modes, Challenge Framing, Query Protocol, Synthesis Notes
