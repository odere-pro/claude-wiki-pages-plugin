## Summary

The obsidian-experience guide covers the Obsidian-specific features of the wiki: the graph view (topic islands, color groups), the wikilink navigation, the Dataview dashboard, and the recommended Obsidian plugin set. It explains how the plugin generates `.obsidian/` config deterministically so the vault opens consistently.

## Key Claims

Recommended Obsidian plugins: Dataview (for dashboard.md queries), obsidian-wikilink-types (50+ pages — for typed link display). The graph view renders topic islands — each top-level topic folder as a colored cluster, with sources/synthesis/index filtered out of the drawn graph. Color groups are deterministic: generated by the polish agent or by running the graph-colors skill. `.obsidian/app.json` and `.obsidian/graph.json` are generated by `apply-obsidian-config.sh`; if lost, re-run the script. Obsidian resolves links by basename/path, never by alias or title — the piped-basename convention (`Title`) is required for correct navigation.
