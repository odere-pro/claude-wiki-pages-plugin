# Design Diagram Template

## Summary

Template file for new design diagrams. Specifies zoom level, perspective, authority link, purpose statement, mermaid fence, reading guide (max 3 bullets), and cross-links. The `[speculative]` marker must be present and removed when all nodes are grounded.

## Key Claims

- Template structure: zoom level, perspective, authority link, one-sentence purpose, mermaid fence, reading guide (≤3 invariants), see-also section.
- All nodes must reference real repo entities; placeholder nodes require `[speculative]` marker.
- Layer names must be Title Case: "Layer 1 — Data", not "layer1" or "data layer".
- Remove `[speculative]` marker only when every node in the diagram is grounded.
