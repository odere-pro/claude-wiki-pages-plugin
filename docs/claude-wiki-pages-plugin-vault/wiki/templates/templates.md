# Templates

The `templates/` directory holds seed files the plugin writes during initialization — primarily `default.config.json`, the baseline plugin configuration.

## Pages

### Concepts

- [[default-config|Default Config]] — the seed configuration written on first enable; defines minimal defaults for `autoHeal`, `gitCheckpoint`, and more; must stay in lockstep with `schemas/config.schema.json`
