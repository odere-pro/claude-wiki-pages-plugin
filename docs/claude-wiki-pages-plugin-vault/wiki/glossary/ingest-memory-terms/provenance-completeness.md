# provenance-completeness

## Definition

The property of a wiki page that every claim traceable to a raw source carries an explicit `sources` entry (and optionally `source_quotes`). Checked by the provenance-completeness lint rule (I3).

## Key Principles

- The property of a wiki page that every claim traceable to a raw source carries an explicit `sources` entry (and optionally `source_quotes`).
- Canonical term in the claude-wiki-pages **Ingest and memory terms** vocabulary; conforms to the project glossary and is enforced by `validate-docs.sh`.

## Examples

- `sources`
- `source_quotes`
