# derived claim

## Definition

A page (or claim) marked `derived: true` (schema v2): LLM inference synthesised across sources rather than stated in one. Carries less direct evidentiary weight; keep `confidence` < 0.8 unless multi-source.

## Key Principles

- A page (or claim) marked `derived: true` (schema v2): LLM inference synthesised across sources rather than stated in one.
- Canonical term in the claude-wiki-pages **Schema terms** vocabulary; conforms to the project glossary and is enforced by `validate-docs.sh`.

- `derived: true`
- `confidence`
