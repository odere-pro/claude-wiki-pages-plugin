# topic folder

## Definition

A folder under `wiki/` holding pages on one subject. Max nesting: four levels.

## Key Principles

- A folder under `wiki/` holding pages on one subject.
- Canonical term in the claude-wiki-pages **Schema terms** vocabulary; conforms to the project glossary and is enforced by `validate-docs.sh`.

## Examples

- `wiki/`
