# ROOT hub

## Definition

The vault MOC `wiki/index.md`, drawn as the graph's single entry-point node — un-hidden from the island filter, given a distinct color group, and aliased `ROOT` — so every topic island visibly hangs off one findable root. See ADR-0033 and the polish agent.

## Key Principles

- The vault MOC `wiki/index.md`, drawn as the graph's single entry-point node — un-hidden from the island filter, given a distinct color group, and aliased `ROOT` — so every topic island visibly hangs off one findable root.
- Canonical term in the claude-wiki-pages **Schema terms** vocabulary; conforms to the project glossary and is enforced by `validate-docs.sh`.
