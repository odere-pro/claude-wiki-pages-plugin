# topic page

## Definition

A page with `type: topic` (schema v2): a narrative landing page for a topic, distinct from the folder's folder note Map of Content.

## Key Principles

- A page with `type: topic` (schema v2): a narrative landing page for a topic, distinct from the folder's folder note Map of Content.
- Canonical term in the claude-wiki-pages **Schema terms** vocabulary; conforms to the project glossary and is enforced by `validate-docs.sh`.

## Examples

- `type: topic`
