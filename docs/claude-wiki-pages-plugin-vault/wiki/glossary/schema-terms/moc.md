# MOC

## Definition

Map of Content. Established PKM term for a navigation page over a scope. Frontmatter `type: index`. Per-folder MOC is the folder note; vault MOC is `wiki/index.md`. Scope differs; the concept is the same.

## Key Principles

- Map of Content.
- Canonical term in the claude-wiki-pages **Schema terms** vocabulary; conforms to the project glossary and is enforced by `validate-docs.sh`.

## Examples

- `type: index`
- `wiki/index.md`
