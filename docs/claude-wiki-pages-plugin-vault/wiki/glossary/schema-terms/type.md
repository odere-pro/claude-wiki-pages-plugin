# type

## Definition

Frontmatter field naming a page's category. One of `source`, `entity`, `concept`, `synthesis`, `index`, `log` (v1); v2 adds `topic`, `project`, `manifest`. The primary filter.

## Key Principles

- Frontmatter field naming a page's category.
- Canonical term in the claude-wiki-pages **Schema terms** vocabulary; conforms to the project glossary and is enforced by `validate-docs.sh`.

## Examples

- `source`
- `entity`
- `concept`
- `synthesis`
- `index`
