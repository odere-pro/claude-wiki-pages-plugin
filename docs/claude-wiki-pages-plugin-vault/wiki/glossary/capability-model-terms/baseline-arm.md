# baseline arm

## Definition

The scaffolding-ablation control arm produced with the generic prompt a user would write without the plugin ("extract the knowledge into well-organized notes", "answer the question from these notes"). Expected to fail the calibrated bar — that gap is the measured value of the scaffolding. Its zero fabrication floor is vacuous (nothing it claims is sourced).
