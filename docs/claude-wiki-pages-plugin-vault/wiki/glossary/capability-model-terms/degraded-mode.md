# degraded mode

## Definition

Operation at a lower capability tier when the full-capability model (Claude) is unavailable. The plugin remains functional for the capabilities of the active tier.

## Key Principles

- Operation at a lower capability tier when the full-capability model (Claude) is unavailable.
