# sync

## Definition

Pull docs changes from a wired source into `raw/` as immutable snapshots, mark superseded source notes, and queue re-ingest. Manual: the heartbeat recommends it, never runs it.

## Key Principles

- Pull docs changes from a wired source into `raw/` as immutable snapshots, mark superseded source notes, and queue re-ingest.
- Canonical term in the claude-wiki-pages **User-facing verbs** vocabulary; conforms to the project glossary and is enforced by `validate-docs.sh`.

- `raw/`
