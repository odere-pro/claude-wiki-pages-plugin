# search

## Definition

Deterministic keyword retrieval over `wiki/`: ranks pages (title/alias > tag > body) and returns `wikilink`-ready hits. A candidate set, not a cited answer. Backed by the engine `search` command.

## Key Principles

- Deterministic keyword retrieval over `wiki/`: ranks pages (title/alias > tag > body) and returns `wikilink`-ready hits.
- Canonical term in the claude-wiki-pages **Layer 2 skills** vocabulary; conforms to the project glossary and is enforced by `validate-docs.sh`.

## Examples

- `wiki/`
- `wikilink`
- `search`
