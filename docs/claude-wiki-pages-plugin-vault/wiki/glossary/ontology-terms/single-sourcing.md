# single-sourcing

## Definition

Storing a fact in exactly one wiki page and referencing it elsewhere via `wikilinks`, so updates propagate without copy-paste drift.

## Key Principles

- Storing a fact in exactly one wiki page and referencing it elsewhere via `wikilinks`, so updates propagate without copy-paste drift.
- Canonical term in the claude-wiki-pages **Ontology terms** vocabulary; conforms to the project glossary and is enforced by `validate-docs.sh`.
