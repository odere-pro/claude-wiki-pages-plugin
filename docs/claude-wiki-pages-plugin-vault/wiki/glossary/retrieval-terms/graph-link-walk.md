# graph link-walk

## Definition

Following typed wikilinks (`sources`, `related`, `depends_on`) from a seed page to its N-hop neighbourhood. The non-embedding recall mechanism; walks the existing Obsidian graph structure.

## Key Principles

- Following typed wikilinks (`sources`, `related`, `depends_on`) from a seed page to its N-hop neighbourhood.
- Canonical term in the claude-wiki-pages **Retrieval terms** vocabulary; conforms to the project glossary and is enforced by `validate-docs.sh`.

## Examples

- `sources`
- `related`
- `depends_on`
