# candidate filter

## Definition

A `--type`, `--folder`, or `--tag` argument that restricts the search corpus before ranking, improving precision. Implemented in the engine `search` command.

## Key Principles

- A `--type`, `--folder`, or `--tag` argument that restricts the search corpus before ranking, improving precision.
- Canonical term in the claude-wiki-pages **Retrieval terms** vocabulary; conforms to the project glossary and is enforced by `validate-docs.sh`.

## Examples

- `--type`
- `--folder`
- `--tag`
- `search`
