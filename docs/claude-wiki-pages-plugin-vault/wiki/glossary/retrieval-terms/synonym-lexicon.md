# synonym lexicon

## Definition

The checked-in file of curated term→alias mappings used to expand a query before the keyword match. Distinct from frontmatter `aliases`, which are page-level; the lexicon is vault-global and governed.

## Key Principles

- The checked-in file of curated term→alias mappings used to expand a query before the keyword match.
- Canonical term in the claude-wiki-pages **Retrieval terms** vocabulary; conforms to the project glossary and is enforced by `validate-docs.sh`.
