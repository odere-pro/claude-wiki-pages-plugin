# graph-traversal primitive

## Definition

The single engine function that executes a graph link-walk given a seed page, a predicate set (from `ontology-profile-v1`), and a hop limit N. Returns scored page references, never page bodies. Shared by R2, R3, and C1.

## Key Principles

- The single engine function that executes a graph link-walk given a seed page, a predicate set (from `ontology-profile-v1`), and a hop limit N.
- Canonical term in the claude-wiki-pages **Retrieval terms** vocabulary; conforms to the project glossary and is enforced by `validate-docs.sh`.
