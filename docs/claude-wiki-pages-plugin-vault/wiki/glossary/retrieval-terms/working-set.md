# working set

## Definition

The subset of wiki pages loaded into the LLM's context window for a given query or agent turn, bounded by the context budget.

## Key Principles

- The subset of wiki pages loaded into the LLM's context window for a given query or agent turn, bounded by the context budget.
- Canonical term in the claude-wiki-pages **Retrieval terms** vocabulary; conforms to the project glossary and is enforced by `validate-docs.sh`.
