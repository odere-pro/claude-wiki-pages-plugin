# NOTICE

## Definition

`NOTICE` at the repo root. Attribution for bundled third-party code. Apache-2.0 requires preservation.

## Key Principles

- `NOTICE` at the repo root.
- Canonical term in the claude-wiki-pages **Authoritative files** vocabulary; conforms to the project glossary and is enforced by `validate-docs.sh`.
