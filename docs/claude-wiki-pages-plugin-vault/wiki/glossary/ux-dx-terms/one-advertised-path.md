# one advertised path

## Definition

The UX principle that exactly one verb is promoted as the entry point for each task (`/claude-wiki-pages:wiki`); alternatives exist but are documented below a fold.

## Key Principles

- The UX principle that exactly one verb is promoted as the entry point for each task (`/claude-wiki-pages:wiki`); alternatives exist but are documented below a fold.
- Canonical term in the claude-wiki-pages **UX and DX terms** vocabulary; conforms to the project glossary and is enforced by `validate-docs.sh`.
