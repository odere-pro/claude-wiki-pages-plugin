# Pandoc — Universal Document Converter

## Metadata
- Author: John MacFarlane

## Summary
Pandoc is a free and open-source document converter written in Haskell by John MacFarlane. It converts files between markup formats such as Markdown, reStructuredText, HTML, LaTeX, and Microsoft Word docx. Pandoc was first released in 2006. It supports exporting to PDF via a LaTeX engine. The project is distributed under the GPL license.
