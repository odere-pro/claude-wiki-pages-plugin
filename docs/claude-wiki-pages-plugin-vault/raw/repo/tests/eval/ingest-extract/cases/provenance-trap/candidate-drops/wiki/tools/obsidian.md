# Obsidian

## Overview

Obsidian is a note-taking application. It exists in this golden reference so
the `provenance-trap` case has an entity whose claims ALL trace back to
[[Obsidian — Notes App]] — and crucially invents no price, license, or revenue
claim the source never stated.

## Key Facts

- Stores notes as local Markdown files.
- Created by Erica Xu and Shida Li; first released in 2020.
- Supports a graph view of links between notes.
