# Obsidian — Notes App

## Metadata
- Author: Erica Xu and Shida Li

## Summary
Obsidian is a note-taking application that stores notes as local Markdown files. It was created by Erica Xu and Shida Li and first released in 2020. Obsidian supports a graph view that visualizes links between notes. Many users extend it with community plugins. This source is referenced by the entity page [[Tools — Index]].
