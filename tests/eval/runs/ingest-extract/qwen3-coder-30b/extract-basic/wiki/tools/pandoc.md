# Pandoc

## Overview
Pandoc is a free and open-source document converter written in Haskell by John MacFarlane.

## Key Facts
- Converts files between markup formats such as Markdown, reStructuredText, HTML, LaTeX, and Microsoft Word docx.
- First released in 2006.
- Supports exporting to PDF via a LaTeX engine.
- Distributed under the GPL license.
