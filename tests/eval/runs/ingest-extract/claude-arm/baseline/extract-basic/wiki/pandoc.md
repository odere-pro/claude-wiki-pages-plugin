# Pandoc

Pandoc is a free and open-source universal document converter written in
Haskell by John MacFarlane.

## Overview

- **What it is:** a document converter between markup formats
- **Author:** John MacFarlane
- **Language:** Haskell
- **First released:** 2006
- **License:** GPL

## Format support

Converts files between markup formats such as:

- Markdown
- reStructuredText
- HTML
- LaTeX
- Microsoft Word docx

PDF export is supported via a LaTeX engine.
