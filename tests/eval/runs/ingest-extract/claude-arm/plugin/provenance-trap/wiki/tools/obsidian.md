# Obsidian

## Overview

Obsidian is a note-taking application that stores notes as local Markdown
files.

## Key Facts

- Created by Erica Xu and Shida Li.
- First released in 2020.
- Supports a graph view that visualizes links between notes.
- Many users extend it with community plugins.
