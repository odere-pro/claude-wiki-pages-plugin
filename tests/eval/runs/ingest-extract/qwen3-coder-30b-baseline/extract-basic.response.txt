# Pandoc

## Overview
- Universal document converter
- Free and open-source software
- Written in Haskell by John MacFarlane
- Converts files between markup formats

## Features
- Supports conversion between multiple markup formats
- Can export to PDF via LaTeX engine
- Distributed under GPL license

## History
- First released in 2006

## Supported Formats
- Markdown
- reStructuredText
- HTML
- LaTeX
- Microsoft Word docx

## Technical Details
- Programming language: Haskell
- License: GPL
