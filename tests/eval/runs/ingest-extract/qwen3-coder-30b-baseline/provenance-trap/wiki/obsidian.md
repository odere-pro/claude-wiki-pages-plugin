# Obsidian — Notes App

## Overview
- Obsidian is a note-taking application that stores notes as local Markdown files
- Created by Erica Xu and Shida Li
- First released in 2020

## Key Features
- Stores notes as local Markdown files
- Supports graph view that visualizes links between notes
- Extensible with community plugins

## Development

## Technical Details
- Uses local Markdown file storage
