# Pandoc

Pandoc is a free and open-source document converter written in Haskell by
John MacFarlane.

## Key facts

- Converts between markup formats: Markdown, reStructuredText, HTML, LaTeX,
  and Microsoft Word docx.
- First released in 2006.
- Supports exporting to PDF via a LaTeX engine.
- Distributed under the GPL license.

See also: [supported formats](formats.md).
